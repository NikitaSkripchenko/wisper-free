A small workflow for selected audio.

Wisper is built for moments when opening a document is too much ceremony. Capture an audio input, let transcription run after stop, then use the text wherever it belongs.

Step 01

Record the source Wisper can hear

Start from the app or the default Command Shift Space shortcut. Pause, resume, discard, or stop from the native overlay.

Step 02

Transcribe after stop

When recording stops, Wisper saves the file locally and sends it to OpenAI for transcription with long-audio chunking when needed.

Step 03

Copy, export, or retry

Use the transcript immediately, reveal the audio in Finder, export text, play it back, or retry a failed transcription.

The honest no-bot path.

The goal is call notes without a meeting guest. The first hard gate is direct system audio plus microphone capture on macOS. Until that ships, Wisper stays clear about what the current release can record.

Available now

Record selected audio input, transcribe after stop, keep local history, copy/export text, and retry failed jobs.

Next milestone

Capture system audio and microphone together, then save one mixed recording that flows into the existing transcript history.

North star

Live transcript during calls, with no bot in the room and no Wisper account in the way.

Feels like a Mac app because it is one.

No web dashboard pretending to be native. Wisper uses SwiftUI, Keychain, native audio input capture, Application Support storage, and system-wide shortcuts.

Native recording overlay

Keep control while recording with Discard, Start Over, Pause, Resume, and Stop.

Long recordings handled locally

AVFoundation splits longer recordings into chunks, then Wisper stitches transcripts back in order.

Useful history, not a black box

Play audio, reveal files, copy transcripts, save text files, retranscribe, or remove entries.

Private where the Mac can keep it private.

Wisper does not require an account. Your OpenAI API key is stored in Keychain. Recordings and local JSON history stay on your Mac. Audio goes to OpenAI for transcription when you stop a recording or retry a transcript.
